As Galileo proved in 1581, the angle has no actual effect on the frequency of the ocsillations (or the pendulum period).
Therefore, with this formula, to find the time for a pendulum period, we will only need to know the value of gravity (g), which is constant as 9.8 meters/s2 on earth, and the pendulum length (L), which is basically the length of the string in which the string is suspended.
Example: If the pendulum length (L) is 50 cm, then if we were to neglect air resistance and/or stupid people waving around their hands, a period would last for around 1.42 seconds (3 s.f.)
